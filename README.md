# Curso USG - Diplomado en Rehabilitación Intervencionista

Plataforma educativa médica de alto rendimiento para el dominio de procedimientos guiados por ecografía.

## Características

- Dashboard Ejecutivo para Administración
- Gestión de Alumnos y Contenido
- Configuración de Certificados
- Reproductor de Doble Perspectiva (Cámara + Ecografía)
- Autenticación con Supabase
- Despliegue en Netlify

## Desarrollo

```bash
npm install
npm run dev
```
